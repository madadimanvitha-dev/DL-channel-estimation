Pilotless Channel Estimation for Chaotic Wireless Communication
↗ GitHub

A deep neural network that estimates wireless channel parameters without pilot signals — recovering channel state from autocorrelation patterns of received signals in highly noisy, chaotic environments.

  • Designed a fully pilotless estimation architecture, eliminating overhead bandwidth waste
  • Pre-trained the model using a Stacked Denoising Autoencoder (SDAE) to learn noise-robust representations from signal autocorrelation
  • Achieved measurable reduction in MSE and improved BER compared to traditional LS and pilot-based baselines
  • Implemented end-to-end pipeline in both MATLAB (simulation) and TensorFlow (model training)
